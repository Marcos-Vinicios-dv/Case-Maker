## case-maker

<p>Esse é o projeto que estou desenvolvendo juntamente com mais 2 alunos do Cotil como TCC, ele ainda não foi finalizado mas sua proposta é um e-commerce de gabinetes de computadores customizados, onde o usuário pode escolher entre alguns modeles de gabinetes e alterar a cor de qualquer parte do gabinete e compra-lo da maneira que foi customizado./p>
  
<p>Futuramente terá um Back-end real como base, uma aplicação desktop para controle de estoque e um app mobile desenvolvido em React Native, ambas as partes estão em desenvolvimento.</P>

## Inicializar projeto

<p>Para inicializar esse projeto em sua máquina será necessário usar os seguinte comandos(como o node instalado):</p>

- Iniciar Back-end (na pasta raiz do projeto + json-server intalado como dependência global) 

    Iniciar server localmente
    ```json-server server.json -p 3333```

- Iniciar Front-end

    Instalar dependências 
    ```npm install```

    Inicializar aplicação localmente
    ```npm run dev```
